# CodeBase Agent 🤖

## AI-Powered Developer Assistant for Code Understanding, Documentation & Test Generation

---

### 🎯 What It Does

| Feature | Description |
|---------|-------------|
| **💬 Chat Q&A** | Ask natural language questions about any GitHub repo |
| **📝 Documentation** | Generate professional markdown docs for any file |
| **🧪 Unit Tests** | Create pytest unit tests for any function instantly |

---

### 🛠️ How It Works

1. **Load** any public GitHub repository
2. **Ask** questions about the codebase
3. **Generate** documentation and tests automatically
